# (Express + JWT) Cascaron

Initial configuration for a NodeJS project with Express and JWT authentication.

1. Installation

```bash
yarn install
```

2. Update dependencies

```bash
yarn upgrade-interactive --latest
```

3. Run

```bash
# Development
yarn start:dev

# Production
yarn start
```

## Socket.io

For test try to connect to

```bash
http://localhost:<port>
```
